# EventHub - User Stories

## Event Creation and Management
* As a regular user, I want to create an event specifying title, description, category, venue, dates, and maximum capacity, so that I can organize activities for the university community.
* As a regular user, I want to edit or cancel the events I have created, so that I can keep the event details accurate and updated.
* As a regular user, I want to view the list of registered attendees for my events and track their registration status, so that I know who is participating.

## Event Discovery and Browsing
* As a regular user, I want to browse a list of all published events and filter them by category, so that I can discover activities that match my interests.
* As a regular user, I want to view the details of an event including venue information and available spots, so that I can decide if I want to participate.

## Registration and Waiting List Behavior
* As a regular user, I want to register for an upcoming event, so that I can secure my spot to attend.
* As a regular user, I want to be placed on a waiting list if an event has reached its maximum capacity, so that I can automatically be promoted if a spot opens up.
* As a regular user, I want to cancel my registration for an event, so that my spot can be given to someone on the waiting list.

## Review and Rating of Past Events
* As a regular user, I want to leave a rating (1-5) and a written comment on completed events I have attended, so that I can help other users decide on future events.

## Administrative Actions (Moderation)
* As an administrator, I want to edit or remove any event, so that I can moderate the platform and ensure it is used properly.
* As an administrator, I want to manage venues, categories, and users, so that the platform remains organized and functional.
